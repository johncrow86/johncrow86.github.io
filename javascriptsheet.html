String Methods
String.length Returns the length of the String
String.starsWith(var) Returns true if the string starts with var
String.toLowerCase() Returns a string in all lower case
String.toUpperCase() Returns a string in all upper case
String.trim() Returns a string removes leading and trailing whitespace
String.split(character) Split the string into an array element at each character
Returns array
Math Methods
Math.random() Returns a number between 0 inclusive and 1 exclusive
Math.floor() Rounds a number down
Math.ceil() Rounds a number up
Array Methods
Array.length Returns the length of the Array
Array.push(item) Add item(s) to the end of an Array
Returns length
Array.pop() Remove the last item from an Array
Returns item
Array.shift() Remove the first item from an Array
Returns Item
Array.unshift(item) Add item(s) to the beginning of an Array
Returns length
Array.join(character) Concatinate all elements of an array into a string seperated by character
Returns string
Array.slice(start position, end position) Return a shallow copy of the list
Array.indexOf(item) Return the index of item

Array.forEach(function) Perform the function on each item in the array
Returns undefined
Array.map(function) Perform the function on each item in the array
Returns new array
Array.filter(function) Checks function boolean and keep true values
Returns new array
Array.findIndex(function) Finds the index of first true value
Returns index
Array.some(function) Returns true if any element meets the functions conditions
Returns boolean
Array.every(function) Returns true if all elements meet the functions conditions
Returns boolean
Object Methods
Object.keys(obj) Returns the objects keys as an array
Object.entries(obj) Returns the objects key/value pairs as an array
Object.assign(target, source) Returns target object with target and source properties

element.removeChild(element) Remove an element
