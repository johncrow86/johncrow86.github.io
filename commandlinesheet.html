Basics

Basic Commands
Command Output
pwd Print the Working Directory
ls Lists all files and directories in the working directory
ls -alt -a lists hidden
-l lists long
-t lists by time
cd <destination directory> Change to the destination Directory
cd .. Change up one Directory level
cd ~ Change to the home Directory
mkdir <directory> Create a Directory
touch <file> Create a File
clear Clear the terminal
tab (button) Autocomplete the name of a directory or file
Basic Commands II
Command Output
echo <string/file> Output the string/file
cat <file> List contents of a file
cp <source file/dir> <new file/dir> Save a copy of the source as a new file
cp <source file/dir> <destination dir> Save a copy of the source in the destination
mv <source file/dir> <new file/dir> Rename the source as a new file
mv <source file> <destination dir> Move the source to the destination
rm <source file> Remove the source file
rm -r <source dir> Remove the source directory
* Wildcard to select all files in a directory
(*.txt will select all txt)

Extra Commands

Git Docs
git init Initialize a GIT Repository to the working directory
git status Print status of working directory and staging area
git add filename Add file to the staging area
git diff filename Print difference between file in working and staging/repo
git commit -m "message" Commits changes from staging area
Leave a message describing changes made
git log Print the history of commits
git show HEAD Print the last commit log and changes
git checkout HEAD filename Checkout most recent commit
Discards changes in working directory if already checked out
git reset HEAD filename Remove file from staging area
Does not discard changes
git reset commit_SHA(first 7) Sets HEAD to commit with commit_SHA
git stash Stash working files in a hidden directory
Frees up the working directory
git stash pop Retrieve working files that have been stashed
