Setup

Doctype

All HTML documents must start with a <!DOCTYPE> declaration.
The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.

HTML

The <html> tag represents the root of an HTML document.
The <html> tag is the container for all other HTML elements (except for the !DOCTYPE tag).

Body

The <body> tag defines the document's body.
The <body> element contains all the contents of an HTML document.
Note: There can only be one <body> element in an HTML document.

Head

The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.
Metadata is data about the HTML document. Metadata is not displayed.
Metadata typically define the document title, character set, styles, scripts, and other meta information.
The <title> element is the tab title.

Basic HTML boiler plate with the <!DOCTYPE> declaration and <html>, <body> and <head> tags.

Basic Elements

Links

The <a> element defines a hyperlink.
The href attribute indicates the links destination.
Set target="_blank" to open in a new window.

Divs

The <div> element defines a division or a section in an HTML document.
It is the basic container for HTML elements and can be easily styled by using the class or id attribute.

Class & ID

Class & ID are attributes added to tags to categorize and identify them.
Class is more broad and can be shared by multiple elements.
ID is specific and should be used to uniquely identify one element.

Paragraphs & Spans

The <p> element will display a block of text.
The <span> element will display inline text.

Basic Elements II

Styling

The <em> element will emphasize.
The <strong> element will bold.

Line Breaks

The <br> tag will create a new line. No closing tag.

Lists

Lists can be unordered <ul> or ordered <ol>.
The <li> element defines an item.

Images

The <img> element defines an image. No closing tag.
The src attribute indicates the images location.
The alt attribute displays text when the image fails to load.

Videos

The <video> element defines a video.
The src attribute indicates the videos location.
The controls attribute displays play/pause.

Advanced Elements

Tables

The <table> element creates a table.
<thead> and <tbody> define the table head and table body.
<tr> creates a table row.
<th> defines a table heading.
scope attribute can be col or row for the column or row headings.
<td> defines table data.
colspan and rowspan attributes can extend the data cell.

Forms

The <form> element creates a form.
action attribute determines where the form is sent.
method attribute determines the type of HTTP request.

The <input> tag accepts user data input.
type attribute defines the input type allowed.
name attribute is the variable name used to submit the input.
required makes the input required.
value attribute is the value of the input.

The <label> element is used to label the input field.
for attribute should match the id attribute of <input>.
Use <input> with type="submit" and value="Submit" to submit the form.
