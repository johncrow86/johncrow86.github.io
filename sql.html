CRUD in SQL

Command Output
Create
Create Table CREATE TABLE table_name (column_name data_type *constraint*);
Create Rows INSERT INTO table_name (column_name)
VALUES (column_values);
Read
Read All SELECT * FROM table_name;
Read SELECT column_name
FROM table_name
WHERE column_name = value;
Update
Add Column ALTER TABLE table_name
ADD COLUMN column_name data_type;
Update Rows UPDATE table_name
SET column_name = value
WHERE column_name = value;
Delete
Delete Rows DELETE FROM table_name
WHERE column_name = value;

Queries

Query Keywords
Command Output
Rename Column SELECT column_name AS new_name;
No repeat values SELECT DISTINCT column_name;
Match One SELECT column_name FROM table_name
WHERE column_name LIKE 'Se_en';
Match Many SELECT column_name FROM table_name
WHERE column_name LIKE 'Se&';
Match Null SELECT column_name FROM table_name
WHERE column_name IS (NOT) NULL;
Match Between SELECT column_name FROM table_name
WHERE column_name BETWEEN value AND value;
ORDER BY SELECT column_name FROM table_name
ORDER BY column_name ASC/DESC;
LIMIT SELECT column_name FROM table_name
LIMIT 5;

Aggregates

Aggregates
Command Output
SELECT COUNT(column) Count the number of rows
SELECT SUM(column) Return the sum
SELECT MAX/MIN(column) Return max/min of column
SELECT AVG(column) Return the average
SELECT ROUND(column) Sum the column
GROUP BY Combine rows of aggregate functions
HAVING(column) Filter results of aggregate functions
